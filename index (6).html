Recursividad en Acción: Factorial

La recursividad es una técnica donde una función se llama a sí misma. Observa cómo funciona con el ejemplo del factorial.

Código


int factorial(int n) {
    if (n <= 1) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

Punteros en C++

Los punteros son una de las características más potentes y complejas de C. Un puntero es una variable que almacena la dirección de memoria de otra variable. Esto permite la manipulación directa de la memoria, la creación de estructuras de datos dinámicas y mucho más.

Conceptos Clave:

  • Declaración: Se declaran con un asterisco (*), como int *p;.
  • Operador de Dirección (&): Obtiene la dirección de memoria de una variable.
  • Operador de Indirección (*): Accede al valor almacenado en la dirección de memoria apuntada.
#include 

int main() {
    int var = 20;
    int *p = &var; // Declaración e inicialización en una línea

    printf("Valor de var: %d\n", var);
    printf("Dirección de var: %p\n", (void*)&var);
    printf("Valor al que apunta p: %d\n", *p);

    *p = 30; // Modificamos el valor de 'var' a través del puntero
    printf("Nuevo valor de var: %d\n", var);

    return 0;
}

👨‍💻 Código:

    //Tenemos la funcion
    1|  void intercambiar(int *a, int *b) {
    2|  int temp = *a;
    3|  *a = *b;

    //En en main
    5|  int x = 10, y = 20;

2| Declaramos temp igual a la direccion donde apunta a.

La función recibe dos punteros a enteros y se utiliza el operador * para acceder a los valores que apuntan y se intercambian usando una variable temporal.

Recursividad

La recursividad es una técnica de programación donde una función se llama a sí misma para resolver un problema. Es especialmente útil para problemas que pueden dividirse en subproblemas más pequeños y similares.

Componentes de una Función Recursiva:

  • Caso Base: Una condición que detiene la recursividad para evitar un bucle infinito.
  • Paso Recursivo: La parte de la función que se llama a sí misma, acercándose al caso base.
#include 

// Función recursiva para calcular el factorial
int factorial(int n) {
    // Caso base: si n es 0, el factorial es 1
    if (n <= 1) {
        return 1;
    }
    // Paso recursivo: n * factorial de (n-1)
    return n * factorial(n - 1);
}

int main() {
    int num = 5;
    printf("El factorial de %d es: %d\n", num, factorial(num));
    return 0;
}

Sistemas Operativos

Un Sistema Operativo (SO) es el software fundamental que gestiona todos los recursos de hardware y software de una computadora. Actúa como intermediario entre el usuario y el hardware de la computadora.

Conceptos Clave:

  • Gestión de Procesos: El SO decide qué proceso se ejecuta y cuándo.
  • Gestión de Memoria: Asigna y libera espacio de memoria para los programas.
  • Sistema de Archivos: Organiza y controla los archivos y directorios.
  • Gestión de E/S: Maneja la comunicación con los dispositivos de entrada y salida.
// Ejemplo conceptual de un planificador simple (pseudocódigo)
void scheduler(process_list) {
    while(true) {
        process p = get_next_process(process_list);
        execute(p);
        if (p.is_finished()) {
            remove_process(process_list, p);
        }
    }
}

Redes de Computadoras

Una red de computadoras es un conjunto de equipos (computadoras, impresoras, etc.) conectados entre sí para compartir información y recursos. Internet es el ejemplo más grande de una red de computadoras.

Modelo OSI:

El modelo OSI es un marco conceptual que estandariza las funciones de un sistema de telecomunicaciones o de computación en siete capas de abstracción.

  • Capa 7: Aplicación - Protocolos de alto nivel como HTTP, FTP, SMTP.
  • Capa 4: Transporte - Proporciona comunicación de extremo a extremo (TCP, UDP).
  • Capa 3: Red - Enrutamiento de paquetes a través de la red (IP).
  • Capa 1: Física - Transmisión de bits crudos sobre un medio físico.

Funciones en C

Las funciones son bloques de código que realizan una tarea específica. Permiten organizar el código, reutilizarlo y hacerlo más legible y mantenible.

Partes de una Función:

  • Tipo de Retorno: El tipo de dato que la función devuelve (int, void, etc.).
  • Nombre: Identificador único de la función.
  • Parámetros: Variables que la función recibe como entrada.
  • Cuerpo: El bloque de código que ejecuta la lógica de la función.
#include 

// Prototipo de la función (buena práctica)
int suma(int a, int b);

int main() {
    int resultado = suma(10, 5);
    printf("El resultado es: %d\n", resultado);
    return 0;
}

// Definición de la función
int suma(int a, int b) {
    return a + b;
}
